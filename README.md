# Motor de Gestión de Gastos y Presupuesto

Este proyecto es un script de JavaScript diseñado para procesar datos financieros, calcular balances en tiempo real y categorizar consumos de forma inteligente.

🌐 Demo en Vivo
Puedes ver y probar la aplicación en funcionamiento entrando al siguiente enlace: 👉 [Ver Gestor de Gastos en Vivo](https://avilasanti.github.io/web_project_expenses_es/)

Tecnologías Utilizadas

- **JavaScript (ES6+)**:
  - Estructuras de arrays.
  - Algoritmos.
  - funciones.
